# Desafio DIO - Automação AWS com CloudFormation

## Descrição
Este projeto tem como objetivo aplicar conceitos de automação de infraestrutura na AWS utilizando o CloudFormation. O foco está na criação e gerenciamento de recursos, como buckets S3 e funções Lambda, de forma automatizada, escalável e documentada.

## Objetivos
- Aplicar conceitos de CloudFormation em um ambiente prático.
- Automatizar a criação de recursos AWS como S3 e Lambda.
- Documentar o processo técnico e compartilhar no GitHub.

## Estrutura do Repositório
- `template.yaml` → Template YAML para CloudFormation.
- `README.md` → Documentação do projeto.
- `lambda/` → Código da função Lambda (se aplicável).

## Passo a Passo
1. Criar um bucket S3 via CloudFormation.
2. Criar uma função Lambda simples.
3. Testar a integração Lambda → S3.
4. Documentar aprendizados e insights.

## Tecnologias
- AWS CloudFormation
- Amazon S3
- AWS Lambda
- GitHub
